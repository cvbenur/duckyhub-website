Background

Système(s) d'exploitation :

Windows

Fonctionnalités :

Change automatiquement le fond d'écran de ta machine vers n'importe quelle image en ligne.

Déroulement du script :

  1. Ouverture du navigateur à la page de l'image
  2. Téléchargement de l'image en PNG
  3. Définition de l'image en tant que fond d'écran

Le script :


            DELAY 3000
            GUI R
            DELAY 100
            STRING iexplore LIEN DE L'IMAGE
            ENTER
            DELAY 2000
            ALT F
            DELAY 100
            STRING a
            DELAY 100
            STRING %USERPROFILE%\PLACEHOLDER.PNG
            DELAY 100
            ENTER
            DELAY 100
            CTRL W
            DELAY 100
            CTRL ESC
            DELAY 100
            STRING paint
            DELAY 100
            ENTER
            DELAY 500 
            CTRL O
            DELAY 100
            STRING %USERPROFILE%\PLACEHOLDER.PNG
            ENTER
            DELAY 300 
            ALT F
            DELAY 100
            STRING k
            DELAY 100
            STRING f
            DELAY 300 
            ALT F
            DELAY 100
            STRING x
