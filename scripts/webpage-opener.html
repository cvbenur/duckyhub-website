Webpage Opener

Système(s) d'exploitation :

Tous

Fonctionnalités :

Ouvre n'importe quelle URL automatiquement.

Déroulement du script :

  1. Ouverture du lanceur Linux
  2. Ouverture du lanceur MacOS
  3. Ouverture du lanceur Windows
  4. Lancement de la page

Le script :


            DELAY 3000
            REM    ###    Lancement pour Linux
            ALT F2
            DELAY 200
            REM    ###    Lancement pour MacOS
            GUI SPACE
            DELAY 200
            REM    ###    Sur Windows, cela change le langage, donc on répète 3 fois
            GUI SPACE
            DELAY 200
            GUI SPACE
            DELAY 200
            GUI R
            DELAY 200
            REM    ###    Sur un autre OS, on pourrait obtenir "   r". On efface donc 4 caractères.
            DELETE
            REPEAT 4
            REM    ###    Lancement de la page.
            STRING [VOTRE URL ICI]
            ENTER
