Script - Shell Looper

Lance la même commande BASH le nombre de fois souhaité.

Vous êtes seul responsable de votre utilisation de ce script.

Système(s) d'exploitation :

Debian Linux (Debian, Ubuntu, Linux Mint, Kali Linux).

Fonctionnalités :

Programme une commande BASH à exécuter en boucle, le nombre de fois souhaité.

Déroulement du script :

  1. Ouverture du terminal
  2. Lancement de la boucle

Le script : 


          DELAY 3000
          REM     ###     Ouverture du terminal
          GUI
          DELAY 450
          REM     ###     Pour Kali Linux, changer la ligne suivante en qterm
          STRING gnome-terminal
          DELAY 250
          ENTER
          DELAY 450
          REM     ###     Lancement de la boucle
          STRING for i in {1..[NBR DE TOURS]}; do [LA COMMANDE ICI]; done
          DELAY 250
          ENTER
