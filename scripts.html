Scripts en blanc

Attention

Tous les scripts présentés ici sont "en blanc", c'est-à-dire qu'ils doivent être modifiés pour pouvoir fonctionner.
Nous vous présentons donc notre extension Visual Studio Code prenant en charge le DuckyScript.

VSCode (Visual Studio Code) est un éditeur de texte publié par Microsoft. Il permet d'éditer du code dans une très large multitude de langages de programmation différents.
Nous recommandons donc son utilisation, couplée à notre extension, pour éditer, écrire et personnaliser vos scripts DuckyScript.

Tous les scripts

La liste complète de tous nos scripts, par ordre alphabétique.

Alias BASH

Programme une nouvelle commande dans le BASH en tant qu'alias.

Auto-Youtube

Règle automtiquement le volume d'un mac à celui souhaité, puis lance n'importe quelle vidéo.

Background

Change automatiquement le fond d'écran de ta machine vers n'importe quelle image en ligne.

Host Redirect

Redirige les liens vers un hôte, par exemple un portail d'entreprise.

iMessage Archiver

Extrait des messages vers un fichier au choix.

Shell Looper

Programme une commande à exécuter en boucle, le nombre de fois souhaité.

Sudo For All

Permet de contourner la nécessité des mots de passe pour un accès administrateur à une machine Apple. Pratique en cas d'oubli de mot de passe.

Webpage Opener

Ouvre n'importe quelle URL automatiquement.

WebRun

Télécharge puis exécute une application depuis Internet.

Windows Update

Affiche une page web imitant un écran de mise à jour Windows.
