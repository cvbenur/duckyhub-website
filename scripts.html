Changer son fond d'écran

Avec ce script, tu pourras faire automatiquement changer le fond d'écran de ta machine en n'importe quelle image en ligne!
Le script ouvre d'abord ton lien, enregistre l'image, l'ouvre dans paint puis la définit en tant que background.

Ouvrir une page Web

Avec ce script, tu pourras ouvrir n'importe quelle URL automatiquement.
Selon ton Operating System, une des trois options se mettra en place. Dans tous les cas, une fenetre s'ouvrira où ton url sera écrite.

Créer une fausse page de mise à jour Windows.

Avec ce script, tu pourras pousser un peu plus loin d'idée du précedent.
Dans cet exemple précis, la machine ouvrira le lien d'un site simulant une mise à jour windows.

Rediriger les sites webs vers un host

Avec ce script, tu pourras rediriger les liens vers un hôte, par exemple un portail d'entreprise.
Ce script altère le driver d'hosting afin d'injecter le lien souhaité.

Executer une application depuis internet

Avec ce script, tu pourras télécharger puis executer les applications de ton choix depuis Internet!
Le script ouvre d'abord une interface de commande, puis génère un vbscript qui pourra, dans la machine, télécharger puis executer un document depuis une n'importe quelle adresse URL.

Sudo pour tous!

Avec ce script, tu pourras contourner la nécessité des mots de passe pour un accès administrateur à ta machine Apple. Pratique quand tu l'as oublié mais que des documents importants s'y trouvent!
Ce script rajoute une ligne dans etc/sudoers qui permet à tout utilisateur d'utiliser l'argument sudo sans avoir à entrer de mot de passe

Auto-Youtube

Avec ce script, tu pourras automtiquement mettre le volume de ton mac à celui souhaité, puis lancer n'importe quelle vidéo!
Le volume est d'abord initialisé, puis le lien est lancé.
